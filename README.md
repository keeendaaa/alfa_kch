# alfa_kch

Интерактивный прототип Alfa Витрины для предпринимателей. Данные демо-сценария хранятся локально в браузере через `localStorage`; реальные платежи и AI-вызовы не выполняются.

## Запуск

Откройте `alfa-vitrina/index.html` в браузере или запустите локальный HTTP-сервер:

```bash
python3 -m http.server 4173 --directory alfa-vitrina
```

Откройте `http://localhost:4173/dashboard.html` и пройдите основной сценарий:

1. Соберите витрину Леры через AI-запуск.
2. В разделе «Рост» включите предоплату 20%.
3. На публичной витрине оформите и оплатите услугу.
4. Проверьте созданные заказ и операцию в кабинете.
